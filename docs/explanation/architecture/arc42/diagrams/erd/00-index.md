# ERD Index (V1)

This folder contains **DBML diagrams** used with dbdiagram.io.

## How to render
1. Open dbdiagram.io
2. Copy the DBML content from the target `.dbml` file
3. Paste into dbdiagram.io to view/export

## V1 diagrams
### System-level
- `system-overview-v1.dbml` — module-boundary overview (ownership + cross-module references)

### Module-level
- `content-v1.dbml`
- `identity-v1.dbml`
- `seo-v1.dbml`
- `media-v1.dbml`
- `interaction-v1.dbml`

### Composition views
- `reading-composition-v1.dbml` — read model join view (no owned tables)
